# Dataset Information

This project uses the **Amazon Reviews - Musical Instruments** dataset, which contains customer reviews and ratings for various musical instruments sold on Amazon.

---

## Dataset Source

- **Kaggle Link:**  
  [Amazon Reviews Full Dataset – Musical Instruments](https://www.kaggle.com/datasets/emaadahmed/amazon-reviews-full-dataset?select=musical_instruments_full.csv)

---

## Description

- **File name:** `musical_instruments_full.csv`  
- **Records:** ~1,048,576 reviews  
- **Columns:** 12  
- **Target variable:** `overall` (the star rating from 1 to 5)  
- **Input variable:** `reviewText` (the customer’s written review)

---

## Data Fields

| Column | Description |
|---------|--------------|
| `reviewerID` | Unique ID of the reviewer |
| `asin` | Product ID |
| `reviewerName` | Name of the reviewer |
| `helpful` | Helpfulness rating (e.g., [2, 3]) |
| `reviewText` | The review text itself |
| `overall` | Star rating (1–5) |
| `summary` | Short summary/title of the review |
| `unixReviewTime` | Time of review (Unix format) |
| `reviewTime` | Date of review |
| `verified` | Whether the purchase was verified |
| `category` | Product category |
| `title` | Product title |
